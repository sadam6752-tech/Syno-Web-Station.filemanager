# File Manager для Synology NAS

**Версия:** v1.0.1

Веб-файловый менеджер для работы с папкой `/upload` на Synology Web Station.

## Установка на Synology NAS

1. Скопируйте все файлы из папки `Dateimanager` в `/volume1/web/`
2. Откройте файл `api.php` и измените строку:
   ```php
   define('BASE_PATH', __DIR__ . '/upload');
   ```
   на:
   ```php
   define('BASE_PATH', '/volume1/web/upload');
   ```
3. Убедитесь, что папка `/volume1/web/upload` имеет права на запись
4. Откройте браузер и перейдите на `http://your-nas-ip/`

## Локальное тестирование

1. Поместите файлы в папку веб-сервера (например, `htdocs/Dateimanager/`)
2. Оставьте в `api.php` строку:
   ```php
   define('BASE_PATH', __DIR__ . '/upload');
   ```
3. Папка `upload` будет создана автоматически в той же директории
4. Откройте `http://localhost/Dateimanager/`

## Структура файлов

- `index.php` - главная страница
- `api.php` - backend API
- `script.js` - frontend логика
- `style.css` - стили
- `.htaccess` - конфигурация Apache

## Функции

- Создание/удаление/переименование файлов и папок
- Загрузка файлов (drag & drop и кнопка)
- Скачивание файлов
- Перемещение файлов между папками (drag & drop)
- Два режима отображения: сетка и список
- Многоязычность: DE, EN, RU
- Адаптивный дизайн
- **Опциональная авторизация** (см. AUTH_SETUP.md)

## Безопасность

- Все операции ограничены папкой `/upload`
- Защита от directory traversal атак
- Валидация путей через `realpath()`
- Опциональная система авторизации с хешированием паролей

## Настройка авторизации

Для включения авторизации см. подробную инструкцию в файле `AUTH_SETUP.md`

Быстрый старт:
1. Откройте `config.php`
2. Установите `AUTH_ENABLED = true`
3. Сгенерируйте хеш пароля через `generate_password.php`
4. Добавьте пользователей в массив `$users`
5. Удалите `generate_password.php`

### Пример добавления пользователя:

```bash
# 1. Откройте в браузере
http://your-nas-ip/generate_password.php

# 2. Введите пароль: MyPassword123
# 3. Скопируйте полученный хеш

# 4. Добавьте в config.php:
$users = [
    'admin' => '$2y$10$...',  // password (по умолчанию)
    'john' => '$2y$10$...',   // MyPassword123 (ваш хеш)
];

# 5. Удалите generate_password.php
# 6. Войдите как john с паролем MyPassword123
```

## Требования

- PHP 7.0+
- Apache с mod_rewrite
- Права на запись в `/volume1/web/upload`
