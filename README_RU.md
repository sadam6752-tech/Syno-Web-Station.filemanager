# Файловый менеджер для Synology NAS

Веб-приложение для управления файлами на Synology NAS через Web Station.

## 🚀 Основные возможности

### Управление файлами
- ✅ Создание, удаление и переименование файлов и папок
- ✅ Копирование файлов и папок с автоматическим переименованием при конфликтах
- ✅ Множественный выбор через чекбоксы
- ✅ Загрузка файлов (drag & drop и через кнопку)
- ✅ Скачивание одного файла или нескольких в ZIP архиве
- ✅ Прогресс-бар загрузки с отображением процента и имени файла

### Перемещение файлов (Drag & Drop)
- ✅ Перетаскивание на папку в списке
- ✅ Перетаскивание на элемент навигации (breadcrumb)
- ✅ Перетаскивание на пустое пространство для перемещения в родительскую папку
- ✅ Кнопка "Переместить вверх" для мобильных устройств
- ✅ Визуальная подсветка целевых элементов

### Поиск и фильтрация
- ✅ Поиск по имени файла в реальном времени
- ✅ Подсветка найденных файлов
- ✅ Скрытие не найденных элементов
- ✅ Очистка поиска кнопкой или клавишей ESC

### Информация и статистика
- ✅ Детальная информация о файлах: размер, даты, права доступа, MIME-тип
- ✅ Для изображений: разрешение и EXIF данные (камера, ISO, выдержка)
- ✅ Статистика выбранных файлов: количество и общий размер
- ✅ Статистика папки: рекурсивный подсчёт файлов, папок и размера

### Визуализация
- ✅ Два режима отображения: сетка и список
- ✅ Миниатюры для изображений (требует PHP GD)
- ✅ Темная тема с сохранением выбора
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Иконки Font Awesome для разных типов файлов

### Многоязычность
- ✅ Немецкий (DE)
- ✅ Английский (EN)
- ✅ Русский (RU)
- ✅ Сохранение выбранного языка

## 🔒 Безопасность

- Работа только в пределах папки `/upload`
- Защита от directory traversal атак
- Проверка всех путей через `realpath()`
- Запрет доступа за пределы BASE_PATH

## 💻 Технические требования

- Synology NAS с Web Station
- Apache + PHP
- PHP расширение GD (опционально, для миниатюр)
- PHP расширение ZipArchive (для скачивания нескольких файлов)

## 📱 Адаптивность

- Десктоп: полный интерфейс с текстом на кнопках
- Планшет/Телефон: компактный интерфейс с иконками
- Toolbar в 3 строки: breadcrumb, статистика/поиск, кнопки
- Оптимизация для сенсорных экранов

## 🎨 Интерфейс

- Современный чистый дизайн
- Плавные анимации и переходы
- Визуальная обратная связь для всех действий
- Подсказки (tooltips) для всех кнопок
- Панель статистики под навигацией
